# 교육관리 플랫폼
**통합 교육 관리 시스템 + RAG 기반 지식 검색 + AI 문제 생성**

## 🎓 프로젝트 개요
FastAPI 백엔드 + Vanilla JavaScript 프론트엔드 + RAG 시스템으로 구성된 통합 솔루션입니다.

- **완성일**: 2026년 01월 08일
- **버전**: v2.6.202601081020
- **상태**: ✅ 프로덕션 배포 완료

## 🎯 최신 업데이트 (v2.6.202601081020)

### ✅ 주요 신규 기능 (2026-01-08)

#### 🔗 URL 공유 미리보기 (Open Graph)
- **소셜 미리보기**: 카카오톡, 페이스북 등에서 URL 공유 시 로고+제목+설명 표시
- **Open Graph 메타 태그**: og:title, og:description, og:image 추가
- **Twitter Card**: 트위터 공유용 메타 태그 추가

#### 📝 신규가입 폼 개선
- **신청 과정 최상단 배치**: 과정 선택을 폼 맨 위로 이동
- **휴대폰 자동 형식화**: 숫자 입력 시 000-0000-0000 형식 자동 변환

#### 📊 대시보드 권한 연동
- **메뉴 권한 기반 표시**: 허용되지 않은 메뉴의 대시보드 위젯 자동 숨김
- **통계 카드**: 학생, 강사, 과정, 시간표, 상담, 프로젝트 카드 권한 체크
- **차트 섹션**: 진로 현황, 강사 분포, 상담 추이 차트 권한 체크
- **빠른 액션 섹션 제거**: 대시보드 UI 간소화

#### 🔐 강사코드 권한 관리 UI 개선
- **계층형 메뉴 구조**: 실제 사이드바 메뉴와 동일한 모양으로 표시
- **테마색 상위 메뉴**: 파란색 그라데이션 헤더로 구분
- **상위 메뉴 연동**: 상위 메뉴 체크 해제 시 하위 메뉴 자동 해제 + UI 숨김
- **권한 없는 메뉴 완전 숨김**: 메뉴 및 상위 드롭다운까지 자동 숨김

#### 🔄 버전 자동 동기화
- **/api/version API**: README.md에서 버전 자동 추출
- **프론트엔드 자동 로드**: 하드코딩 대신 API에서 버전 가져옴

#### 📝 온라인시험/과제 시스템 (완성)
- **시험 유형**: 시험, 퀴즈, 과제 3가지 유형 지원
- **과제 제출**: 파일 첨부 + 텍스트 답변 지원
- **드래그앤드롭**: 과제 파일 업로드 시 클릭/드래그앤드롭 모두 지원
- **마감일 관리**: 과제별 마감일 설정 (datetime-local)
- **재제출 가능**: 마감 전까지 학생이 과제 수정 가능
- **수동 채점**: 과제는 강사가 직접 점수/피드백 입력
- **자동 채점**: 객관식/단답형 문제는 자동 채점
- **강제 종료 처리**: 강사가 시험 종료 시 학생에게 알림 후 목록으로 이동

#### 📢 공지사항 분류 시스템
- **전체 공지**: 모든 학생에게 표시
- **과정별 공지**: 해당 과정 학생에게만 표시
- **교과목별 공지**: 해당 교과목 수강 학생에게만 표시

#### 🤖 AI 로딩 애니메이션
- **AI 전용 로딩**: 문제 생성 시 로봇 아이콘 + 단계별 메시지
- **진행 상태 표시**: "문제 구성 중..." → "최종 검토 중..." → "거의 완료..."

#### 🔍 검색 디바운스
- **학생 검색**: 300ms 디바운스 적용 (타이핑 중 끊김 방지)
- **강사 검색**: 300ms 디바운스 적용

#### 🎨 UI/UX 개선
- **학생 메뉴명 변경**: "온라인시험" → "온라인시험/과제"

- **신규가입 학력**: "학력(학력무관선발가능)" 라벨로 변경
- **과제 카드 UI**: 오렌지색 테마, 마감일/제출상태 표시

#### 🔄 자동 마이그레이션

# 시스템 운영 참고 사항
## 시스템 관리

### DB 마이그레이션 (필요시)
```bash
mysql -h [DB_HOST] -u [DB_USER] -p[DB_PASSWORD] minilms < migrations/0003_add_menu_permissions.sql
```

### 서비스 재시작
```bash
systemctl restart minilms
```

### 서비스 관리
```bash
systemctl start minilms    # 시작
systemctl stop minilms     # 중지
systemctl restart minilms  # 재시작
systemctl status minilms   # 상태 확인
journalctl -u minilms -f   # 로그 실시간 확인
```
